# End-to-End-ML-Project

Workflows

    update config.yaml
    update schema.yaml
    update params.yaml
    update the entity (notebook)
    update the configuration manager in src config (constant->init file)
    update the components
    update the pipeline
    update the main.py
    update the app.py

How to run?
STEPS:

conda create -n venv python=3.8 -y 

conda activate venv/

pip install -r requirements.txt

python app.py
